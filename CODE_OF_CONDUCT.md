# Code of Conduct

This project is about building **kind software**: tools that help people directly, or help people through education.

This repository is not a large community project, but if you interact here (issues, PRs, discussions), please keep it respectful and constructive.

## Expectations

- Be polite and assume good intent.
- Give feedback on code and ideas, not on people.
- Keep discussions practical and calm.
- Respect privacy and do not post someone else's private information.

## Not Acceptable

- Harassment, hate speech, threats, or personal attacks.
- Trolling, repeated hostility, or bad-faith behavior.
- Sexual or abusive language.

## Enforcement

Maintainers may edit or remove comments, close threads, or block contributors when behavior harms the project or people in it.
